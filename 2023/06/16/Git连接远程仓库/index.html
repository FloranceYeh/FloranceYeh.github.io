Git连接远程仓库

2023/6/16 Git

写给机房的儿子们看

我这里写了个自动生成代码，填一下表单然后提交就行

在这里输入Github的名称
在这里输入Github的邮箱
在这里输入Github远程仓库ssh地址

  1. 下Git

  2. 配置用户数据
    git config --global user.name "FloranceYeh"
    git config --global user.email "Florance_Sunrise@outlook.com"

  3. 生成 ssh key
    ssh-keygen -t rsa -C "Florance_Sunrise@outlook.com"

  4. Github 链接 ssh key

    1. cat ~/.ssh/id_rsa.pub
    2. https://github.com/settings/keys to

  5. 检查链接

    1. ssh -T git@github.com
    2. 若22端口失败，配置443端口
      1. vim ~/.ssh/config
      2. 编写 config 里的内容
                    
Host github.com
    User Florance_Sunrise@outlook.com
    Hostname ssh.github.com
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_rsa
    Port 443


                    
bash

  1. 再次检查链接

    1. ssh -T git@github.com

  2. 生成Git文件夹
    git init

  3. 连接仓库
    git remote add origin git@github.com:FloranceYeh/blog.git

  4. 添加文件
    git add .
    .是添加所有文件

  5. 添加注释
    git commit -m "push"

  6. 上传！！！
    git push origin master --force
    –force是强制覆盖上传
